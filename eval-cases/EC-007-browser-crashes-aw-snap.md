# EC-007: Browser crashes (“Aw, Snap”)

## Summary
Chrome crashes or shows “Aw, Snap” due to memory exhaustion or extension issues.

## Reproduction
1. Trigger a crash or capture crash IDs.
2. Collect chrome debug logs or crash reports.
3. Ask why pages are crashing.

## Conversation
User: “We see ‘Aw, Snap’ crashes across multiple devices.”

## Expected result
- Diagnosis suggests memory/extension/profile issues.
- Evidence references crash IDs or debug logs.
- Next steps include disabling extensions and collecting logs.

## Cleanup
- None.
