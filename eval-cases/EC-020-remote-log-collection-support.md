# EC-020: Remote log collection and remote support

## Summary
Admins need to collect logs or provide remote support without physical access.

## Reproduction
1. Attempt log export via Admin console.
2. Verify required privileges.
3. Ask how to enable remote log collection/support.

## Conversation
User: “How can we collect logs remotely and assist users?”

## Expected result
- Diagnosis references Admin console log export flow.
- Evidence includes required privileges or settings.
- Next steps explain remote log collection and Chrome Remote Desktop usage.

## Cleanup
- None.
