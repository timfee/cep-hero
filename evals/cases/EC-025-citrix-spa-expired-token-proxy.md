# EC-025: Citrix SPA proxy pop-up due to expired token

## Summary

Citrix SPA shows proxy pop-ups because CEP token has expired.

## Reproduction

1. Allow token to expire in Citrix SPA.
2. Observe proxy prompt.
3. Ask how to resolve.

## Conversation

User: “Citrix SPA keeps showing proxy pop-ups.”

## Expected result

- Diagnosis identifies expired CEP token.
- Evidence references Citrix logs or token expiry.
- Next steps advise re-authentication and restart.

- None.
