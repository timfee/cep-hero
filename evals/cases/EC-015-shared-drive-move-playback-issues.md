# EC-015: Shared drive move / playback issues

## Summary

File manager cannot move files to shared drive or playback fails due to storage
or outdated ChromeOS version.

## Reproduction

1. Use a device with low storage or older OS version.
2. Attempt file move or playback.
3. Ask why the operation fails.

## Conversation

User: “Can’t move files to shared drive and playback fails.”

## Expected result

- Diagnosis points to storage limits or outdated OS.
- Evidence includes device storage and OS version fields.
- Next steps recommend freeing storage and updating ChromeOS.

## Cleanup

- None.
