# EC-041: Recommended vs mandatory policies

## Summary

Policies are set as RECOMMENDED, allowing users to override them.

## Reproduction

1. Set a policy as Recommended (e.g., homepage).
2. User changes the setting locally.
3. Ask why the policy “isn’t working.”

## Conversation

User: “Why can users still change the homepage policy?”

## Expected result

- Diagnosis explains RECOMMENDED vs MANDATORY behavior.
- Evidence includes policy level from resolve output.
- Next steps suggest switching to mandatory if required.

## Cleanup

- None.
