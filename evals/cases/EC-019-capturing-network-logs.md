# EC-019: Capturing network logs for troubleshooting

## Summary

Admins need net logs to diagnose connectivity or proxy issues.

## Reproduction

1. Run `chrome://net-export` or `--log-net-log`.
2. Capture and review net log file.
3. Ask how to collect or interpret network logs.
