# EC-008: Browser performance problems (high CPU/slow tabs)

## Summary

Chrome performance degrades due to high-CPU tabs or extensions.

## Reproduction

1. Observe high CPU usage via Task Manager/telemetry.
2. Capture performance traces.
3. Ask why Chrome is slow.

## Conversation

User: “Chrome is slow and CPU spikes on some tabs.”

## Expected result

- Diagnosis references telemetry or task manager data.
- Evidence includes CPU usage or extension IDs.
- Next steps recommend disabling offending extensions or tabs.

## Cleanup

- None.
